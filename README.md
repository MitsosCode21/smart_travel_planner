## 1. Απαιτήσεις Συστήματος (Prerequisites)
Για την εκτέλεση του πηγαίου κώδικα, απαιτείται η εγκατάσταση λογισμικού διακομιστή (Web Server) που υποστηρίζει PHP και MySQL.

- **Προτεινόμενο Λογισμικό:** XAMPP (περιλαμβάνει Apache Web Server, PHP 8.x και MariaDB/MySQL).
- **Λήψη XAMPP:** [https://www.apachefriends.org/](https://www.apachefriends.org/)
- **Πρόγραμμα Περιήγησης (Browser):** Google Chrome ή Mozilla Firefox (συνιστάται).
- **Σύνδεση στο Internet:** Απαιτείται για τη φόρτωση εξωτερικών βιβλιοθηκών (CDN) και APIs (Καιρός, Χάρτες).

## 2. Βήματα Εγκατάστασης (Installation Guide)

### Βήμα 2.1: Τοποθέτηση Πηγαίου Κώδικα
1. Εγκαταστήστε και ανοίξτε το **XAMPP Control Panel**.
2. Πατήστε το κουμπί **Start** δίπλα στο `Apache` (Web Server) και στο `MySQL` (Βάση Δεδομένων).
3. Μεταβείτε στον φάκελο εγκατάστασης του XAMPP, συνήθως στο: `C:\xampp\htdocs\`
4. Τοποθετήστε ολόκληρο τον φάκελο του project (με όνομα `smart_travel_planner`) μέσα στον φάκελο `htdocs`.
   *(Η τελική διαδρομή πρέπει να είναι: `C:\xampp\htdocs\smart_travel_planner`)*

### Βήμα 2.2: Εισαγωγή της Βάσης Δεδομένων
1. Ανοίξτε τον browser της επιλογής σας και πληκτρολογήστε τη διεύθυνση: [http://localhost/phpmyadmin/](http://localhost/phpmyadmin/)
2. Στο αριστερό μενού, κάντε κλικ στο **"New"** (Νέα) για να δημιουργήσετε μια νέα βάση.
3. Στο πεδίο "Database name", πληκτρολογήστε **ακριβώς**: `smart_travel_planner`
4. Στο αναπτυσσόμενο μενού επιλέξτε το Collation: `utf8mb4_general_ci` και πατήστε **Create**.
5. Αφού δημιουργηθεί η βάση, επιλέξτε την από το αριστερό μενού.
6. Στο πάνω μενού, πατήστε την καρτέλα **"Import"** (Εισαγωγή).
7. Πατήστε **"Choose File"** και βρείτε το αρχείο `smart_travel_planner.sql` που βρίσκεται μέσα στο φάκελο `database/` του πηγαίου κώδικα.
8. Πατήστε το κουμπί **"Import"** στο κάτω μέρος της σελίδας. 
*(Αν εμφανιστεί πράσινο μήνυμα επιτυχίας, η βάση έχει φορτωθεί σωστά).*

### Βήμα 2.3: Εκτέλεση της Εφαρμογής
1. Ανοίξτε μια νέα καρτέλα στον browser σας.
2. Πληκτρολογήστε τη διεύθυνση: [http://localhost/smart_travel_planner/](http://localhost/smart_travel_planner/)
3. Η εφαρμογή είναι πλέον έτοιμη για χρήση.

## 3. Κωδικοί Διαχειριστή (Admin Credentials)

Για τον έλεγχο των διαχειριστικών λειτουργιών (Admin Panel) και των βάσεων δεδομένων, έχει δημιουργηθεί ένας προεπιλεγμένος λογαριασμός. Για να συνδεθείτε, πηγαίνετε στη σελίδα [Σύνδεσης (Login)](http://localhost/smart_travel_planner/auth/login.php) και χρησιμοποιήστε τα παρακάτω:

- **Email Διαχειριστή:** `admin@smarttravel.com`
- **Κωδικός Πρόσβασης:** `admin123`

*(Οι κωδικοί των απλών χρηστών είναι κρυπτογραφημένοι με αλγόριθμο Bcrypt στη βάση. Μπορείτε να δημιουργήσετε έναν νέο απλό χρήστη μέσω της Φόρμας Εγγραφής της εφαρμογής).*

## 4. Ρυθμίσεις Σύνδεσης (Configuration)

Σε περίπτωση που η τοπική σας MySQL χρησιμοποιεί κωδικό πρόσβασης (στο XAMPP συνήθως είναι κενό), μπορείτε να αλλάξετε τις ρυθμίσεις σύνδεσης (host, username, password) στην κορυφή των παρακάτω κρίσιμων αρχείων:
- `process.php`
- `profile.php`
- Όλα τα αρχεία του φακέλου `admin/`
- Όλα τα αρχεία του φακέλου `auth/`

```php
$host = 'localhost';
$db   = 'smart_travel_planner';
$user = 'root';
$pass = ''; // Αφήστε κενό για το προεπιλεγμένο XAMPP
```

## 5. Δομή Πηγαίου Κώδικα (Source Code Structure)

Ο κώδικας ακολουθεί τη Modular αρχιτεκτονική Separation of Concerns.

- `/` (Root): Τα βασικά αρχεία πλοήγησης (`index.php`) και ο αλγόριθμος επεξεργασίας κρατήσεων (`process.php`).
- `admin/`: Τα εκτελέσιμα αρχεία διαχείρισης συστήματος και το Dashboard (RBAC προστασία).
- `auth/`: Τα αρχεία για Σύνδεση, Εγγραφή και Αποσύνδεση.
- `database/`: Το αρχείο εξαγωγής της βάσης (`.sql`).
- `pages/`: Οι υποσελίδες του συστήματος (Ροή Κράτησης 4 βημάτων, Προορισμοί, Επικοινωνία).
