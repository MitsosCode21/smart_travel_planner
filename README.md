## Περιγραφή

Διαδικτυακή πλατφόρμα που λειτουργεί ως εξατομικευμένος ταξιδιωτικός σύμβουλος. Ο χρήστης εισάγει τα κριτήριά του (budget, ημέρες, άτομα, τύπος διακοπών, τοπίο) και ο αλγόριθμος φιλτράρει προορισμούς, υπολογίζει κόστος και προτείνει ολοκληρωμένα ταξιδιωτικά πακέτα. Περιλαμβάνει σύστημα κρατήσεων 4 βημάτων, αυτόματη έκδοση E-Tickets, μετεωρολογικά δεδομένα πραγματικού χρόνου και Admin Panel.

---

## Τεχνολογίες

**Backend:** PHP 8.x | **Βάση:** MySQL/MariaDB (PDO) | **Frontend:** HTML5, CSS3, JavaScript  
**Εξωτερικά:** Google Fonts (Inter), Chart.js, html2canvas, wttr.in Weather API  
**Server:** XAMPP (Apache + MySQL)

---

## Εγκατάσταση

1. Εγκαταστήστε το **XAMPP** από [apachefriends.org](https://www.apachefriends.org)
2. Τοποθετήστε τον φάκελο `smart_travel_planner` στο `C:\xampp\htdocs\`
3. Ανοίξτε το XAMPP Control Panel → **Start** Apache και MySQL
4. Ανοίξτε [http://localhost/phpmyadmin](http://localhost/phpmyadmin)
5. Δημιουργήστε νέα βάση με όνομα **`smart_travel_planner`** (collation: `utf8mb4_general_ci`)
6. Επιλέξτε τη βάση → **Import** → επιλέξτε το αρχείο `database/smart_travel_planner.sql` → **Go**
7. Ανοίξτε στον browser: **http://localhost/smart_travel_planner/**

## Βασικές Λειτουργίες

**Χρήστης:**
- Αναζήτηση προορισμών με αλγόριθμο budget (αυτόματο fallback αν δεν βρεθούν αποτελέσματα)
- Κράτηση σε 4 βήματα: Κατάλυμα → Μεταφορικό → Στοιχεία → Πληρωμή
- Εποχιακή τιμολόγηση (0.75x – 1.6x) και προσομοίωση διαθεσιμότητας
- Μετεωρολογικά δεδομένα σε πραγματικό χρόνο (wttr.in API)
- Αυτόματη έκδοση E-Ticket (λήψη ως PNG)
- Δίγλωσση υποστήριξη (Ελληνικά / Αγγλικά)

**Διαχειριστής:**
- Dashboard με στατιστικά και γράφημα εσόδων ανά μήνα
- Πλήρης CRUD διαχείριση: προορισμοί, ξενοδοχεία, μεταφορικά, χρήστες, κρατήσεις

## Ασφάλεια

- Κρυπτογράφηση κωδικών: `password_hash()` / `password_verify()` (bcrypt)
- Προστασία SQL Injection: PDO Prepared Statements
- Προστασία XSS: `htmlspecialchars()` σε όλα τα εμφανιζόμενα δεδομένα
- Session Guards σε προστατευμένες σελίδες και έλεγχος ρόλου admin
- Τα δεδομένα κάρτας δεν αποστέλλονται στον server (client-side validation)

## Σημειώσεις

- Οι πληρωμές λειτουργούν σε **κατάσταση προσομοίωσης** (δεν υπάρχει πραγματικό payment gateway)
- Η διαθεσιμότητα ξενοδοχείων είναι **προσομοιωμένη** (ντετερμινιστικός αλγόριθμος)
- Απαιτείται σύνδεση στο Internet για CDN βιβλιοθήκες και weather API
- Τα εργαλεία και οι βιβλιοθήκες που χρησιμοποιούνται είναι ελεύθερου/ανοιχτού λογισμικού

| Πρόβλημα | Λύση |
|----------|------|
| Σφάλμα σύνδεσης βάσης δεδομένων | Βεβαιωθείτε ότι το MySQL τρέχει στο XAMPP Control Panel |

| Δεν φορτώνει το γράφημα στο Admin | Απαιτείται σύνδεση στο Internet για τη φόρτωση του Chart.js μέσω CDN |
